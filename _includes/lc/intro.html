Library Carpentry / Biblioteksløyd er kurs av og for bibliotekarer og andre som jobber i og med bibliotek. Dette kurset passer for deg som

  • Er nysgjerrig på programmering og trenger en dytt for å komme i gang
  • Jobber med (store) datasett.
  • Ønsker å lære nye verktøy for a rydde i, sortere, søke i og kna datasett
  • Ønsker å kunne automatisere kjedelige PC-oppgaver (en vakker dag)

Metodologien bygger på Library Carpentry. For mer info om hva vi lærer bort og hvorfor, se artikkelen "Library Carpentry: software skills training for library professionals".
